\doxysubsection{Detailed Description}
Класс для представления суточного прогноза погоды. 

Класс хранит данные о погоде за одни сутки, включая дату, температуры утром, днем и вечером, погодное явление и количество осадков. Также включает методы для проверки корректности данных.
